# gptengineer-connector

Make a frontend web application tailored to interacting with gptengineer.app by using github commits to upload additional files into a gptengineer project repo. You may expand on this idea to improve the project idea.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/gptengineer-connector.git
cd gptengineer-connector
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.

## Requirements

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
